[END_PERSONA]
<!-- Focus on semantic completeness and stable machine-parseable output -->
[DEFINE_VARIABLES:]
_输入模板: text "Target template structure"
_输入代码: text "Method code to convert"
_子分析结果: list "Sub-code analysis results"
_输出: text "Final JSON string"
[END_VARIABLES]

[DEFINE_WORKER: ConvertMethodToTemplate]
[INPUTS]
<REF> _输入模板 </REF>: text
<REF> _输入代码 </REF>: text
<REF> _子分析结果 </REF>: list "List of sub-code analyses. Example format: [{'类型': 'For', '代码片段': 'for i in range(10):', '分析结果': '...'}]"
[END_INPUTS]

[OUTPUTS]
<REF> _输出 </REF>: text
[END_OUTPUTS]

[MAIN_FLOW]
[SEQUENTIAL_BLOCK]
[COMMAND "Understand the target SPL template structure but do not output any SPL text" RESULT _template_understanding]
[COMMAND "Understand the full method semantics from the input code and all sub-analyses; do not lose any meaningful behavior. IMPORTANT: If any sub-analysis text contains <SPL>...</SPL>, treat it as a required reference marker and preserve it verbatim in any final command/condition/log text that mentions that function name." RESULT _method_understanding]
[COMMAND "Derive a granular step plan where one step equals one coherent process; preserve numeric constants, condition boundaries, early returns, short-circuits, and exception paths. IMPORTANT: When describing a step that calls a referenced internal function, keep the function name wrapped exactly as <SPL>...</SPL> if it appears that way in sub-analyses." RESULT _step_plan]
[COMMAND "Generate brief_description as one single English sentence summarizing the method’s primary purpose and key transformation; it must be a real summary of this code, not a placeholder text" RESULT _brief_summary]
[COMMAND "Construct a strict JSON object following the schema rules; omit optional sections if they do not exist. IMPORTANT: In JSON fields main_flow[].command (and alternative_flows[].steps[].command / exception_flows[].condition / log when applicable), if you mention a function name that is marked as <SPL>...</SPL> in sub-analyses, you MUST keep the <SPL>...</SPL> wrapper exactly; do not remove or alter it." RESULT _json_draft]
[COMMAND "Output only the JSON object as the final answer; no markdown, no comments, no surrounding text" RESULT <REF> _输出 </REF>]
[END_SEQUENTIAL_BLOCK]
[END_MAIN_FLOW]
